## Modpacks are here

You can now publish **modpacks** on Modifold - collections of mods in a single archive (RAR / ZIP). You can upload them, and users will be able to download and install them.

What already works:

- A separate modpacks section next to mods
- The **Modpack** project type in the project creation modal
- Separate tags for modpacks
- Otherwise, the same features as projects of type **Mod**

What does not work yet:

- We don't know yet, but we'd be happy if you tell us about bugs and feature requests you find

Important: **the maximum size for a single version file is 100 MB**. The limit will be increased in upcoming updates.

## Interface and search updates

Highlights:

- New tag selector in project settings: icon buttons instead of a dropdown
- Search page improvements: a **Modpacks** tab and a correct indicator
- Updated background for the mods and modpacks sections
- Redesigned notifications page
- Unauthenticated users can now change the site language and theme
- Fixes and improvements in the mobile version *(it looks great, check it out!)*

## Other improvements

- Updated the settings pages (project and profile) and their side panels
- Added two-factor authentication (2FA). You can enable it in **Profile settings**, under **Security**
- Improved the project comments page
- Tweaked the verification application page
- Organizations can now add social media links
- Moved a person's social media links into a separate block on the profile page
- Removed scripts that were slowing down page loading
- The profile page now lets you open followers and following lists in a modal

## What's next

We'll keep developing modpacks and improving platform stability. Your feedback and criticism are what help us get better.

Thanks for building Modifold together with us.
